How to use
----------

[Click Here](http://guarded-eyrie-8904.herokuapp.com/) for link to site

Clone the repository:
```shell
$ git clone git@github.com:scully87/Pokemon-game.git
```

Change into the directory:
```shell
$ cd Pokemon-game
```

To view locally:
```shell
$ rackup
 - view in broswer at http://localhost:4567
```
